## Phase 1: Configure

### Prerequisite: Install HVE Core - All

This walkthrough uses agents from several HVE Core extension packs.
Install **HVE Core - All** from the marketplace to get every agent
referenced in the 7 steps:

[Install HVE Core - All](command:workbench.extensions.search?%22ise-hve-essentials.hve-all%22)

### Set your partner context first

Click **Set Partner Context** to describe your current partner engagement
(e.g., "Contoso is building a customer support chatbot using Azure OpenAI
and Azure AI Search for RAG over their insurance claims documents").

This context automatically threads through all 7 steps. Instead of
generic sample prompts, every step generates prompts tailored to your
specific partner scenario.

### Then set your identity

The Memory agent stores your role, tech stack, and partner context
so every Copilot interaction starts with the right background.

### What happens

1. You describe your partner engagement (once)
2. You tell Copilot your role and focus areas
3. Every subsequent step uses your partner context to generate relevant prompts
4. You can edit any prompt before sending it

### Time: ~30 seconds
