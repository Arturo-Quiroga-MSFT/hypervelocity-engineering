## Phase 2: Deploy

The IaC Generator produces deployable Bicep or Terraform code
from a description of the Azure resources your partner needs.

### What happens

1. You describe the required resources and configuration
2. The agent generates Bicep with proper dependencies, naming, and best practices
3. You get deployment-ready code with parameter files

### Time: ~5 minutes

Click **Generate IaC** below to open Copilot Chat with a sample
resource description.
