# HVE Core PSA Guidance

This directory contains all guidance materials for Partner Solutions Architects (PSAs) using [HVE Core](https://github.com/microsoft/hve-core) across the GCPS Partner Technical Deliverables lifecycle.

## Contents

### Quick Start Series (under 25 minutes total)

Seven focused, beginner-friendly guides that each solve a real daily problem with minimal effort:

| # | Guide | What You Get | Time |
|---|---|---|---|
| 1 | [Set Your Context Once](quick-starts/hve-quick-start-1-memory.md) | Persistent context forever | 30 sec |
| 2 | [Prep for a Partner Call](quick-starts/hve-quick-start-2-researcher.md) | Structured briefing | 2 min |
| 3 | [Turn Your Whiteboard into a Diagram](quick-starts/hve-quick-start-3-architecture-diagram.md) | Shareable Mermaid diagram | 3 min |
| 4 | [Document Your Architecture Decisions](quick-starts/hve-quick-start-4-adr.md) | Formal decision record | 2 min |
| 5 | [Build a Partner Demo Dashboard](quick-starts/hve-quick-start-5-demo-dashboard.md) | Runnable Streamlit app | 5 min |
| 6 | [Scaffold Azure Resources as Code](quick-starts/hve-quick-start-6-iac-generator.md) | Deployable Bicep/Terraform | 5 min |
| 7 | [Review Code for Security](quick-starts/hve-quick-start-7-security-review.md) | Severity-graded findings | 5 min |

### GCPS How-To Workflows

| Guide | GCPS Stage | PSA Time | Deliverables Covered |
